# Fraud Detection System (Intrusion Detection System)

## Overview
This project is an end-to-end **Fraud Detection / Intrusion Detection System** designed to analyze online payment transactions and identify high-risk or fraudulent behavior using machine learning techniques.

The system allows users to upload datasets, train detection models, analyze transactions, and generate reports for audit and security analysis.

## Features
- Upload and analyze transaction datasets
- Machine Learning–based fraud detection
- High-risk transaction identification
- User-specific dashboards and activity tracking
- PDF report generation
- Secure authentication system

## Technologies Used
- Python
- Flask
- SQLite / SQLAlchemy
- Scikit-learn
- Pandas, NumPy
- HTML, CSS, Bootstrap
- Git & GitHub

## Use Case
Designed for **cybersecurity**, **financial fraud analysis**, and **intrusion detection** scenarios.
